# React Flexboxgrid

http://flexboxgrid.com/ ported to react components.

## Installation

npm install react-flexboxgrid

## Usage

```react
<GridRow>
  <GridColumn xs={12} md={6}>
    Grid Column!
  </GridColumn>
  <GridColumn xs={12} md={6}>
    Grid Column!
  </GridColumn>
</GridRow>
```

## GridRow Props (Defaults)
```
reverse : false
```

## GridColumns Props (Defaults)
```
xs: null,
sm: null,
md: null,
lg: null
```
